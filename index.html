About the Website

This project was developed by Diego Gonzalez Reyes.

Github repo.

Pit Stop Central is a sleek and interactive web application designed for Formula 1 enthusiasts. This platform enables users to explore detailed information about F1 seasons, including races, drivers, constructors, and circuits. It also allows users to manage their favorite items and interact with dynamically generated data in an engaging manner.

    Technologies used:
  • HTML5 and CSS3 (Tailwind CSS for modern styling and responsive design)
  • Vanilla JavaScript for dynamic functionalities and user interactions
    • Backend API:
  • Data sourced from external APIs providing real-time F1 statistics
    • Browser Storage:
  • LocalStorage for persisting user favorites and caching data
